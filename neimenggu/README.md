# neimenggu

内蒙古自治区位于中国北部边疆，由东北向西南斜伸，呈狭长形，东西直线距离2400公里，南北跨度1700公里，横跨东北、华北、西北三大区。土地总面积118.3万平方公里，占全国总面积的12.3%，在全国各省、市、自治区中名列第三位。东南西与8省区毗邻，北与蒙古国、俄罗斯接壤，国境线长4200公里。
