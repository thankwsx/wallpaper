新疆古称西域，自古以来就是祖国不可分割的一部分。公元前138年，汉武帝派张骞出使西域，西汉政权与西域各城邦建立了联系。公元前60年，西汉政权在乌垒（今轮台县境内）设立西域都护府，自此西域正式列入汉朝版图。清乾隆后期改称西域为新疆，1884年正式建立新疆省，省会迪化（今乌鲁木齐市）。1949年新疆和平解放，1955年10月1日成立新疆维吾尔自治区，首府设在乌鲁木齐市（蒙古语意为优美的牧场）。
