http://www.gd.gov.cn/

广东省位于中国大陆最南部，陆地面积17.98万平方公里，海岛面积1592.7平方公里，南临南海，大陆海岸线总长3368.1公里，岛屿众多。全省地处低纬度，北回归线横贯陆地中部。2122公里长的珠江为中国第三大河流，珠江三角洲土地肥沃，是著名的鱼米之乡。
