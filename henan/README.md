河南省位于中国中东部，黄河中下游，黄淮海平原西南部，河南省古代辖区位于黄河之南，故名"河南"。2000多年前，河南是中国九州中心的豫州，所以，河南简称"豫"，且有"中州"、"中原"之称。河南与河北、山西、陕西、湖北、安徽、山东毗邻。全省总面积16.7万平方公里。黄河流经河南境内700多公里。
