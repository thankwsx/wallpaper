# taiwan



台湾自古就是中国神圣领土不可分割的一部分。1949年以来，由于中国内战延续和外部势力干涉，海峡两岸陷入长期政治对立的状态。尽管台湾与祖国大陆尚未统一，但中国主权和领土从未分割，两岸同属于一个中国的事实从未改变。
