# hebei

河北省

河北省地处华北平原，兼跨内蒙古高原。全省中环首都北京和北方重要商埠天津市，北与辽宁、内蒙古为邻，西靠山西，南与河南、山东接壤，东临渤海。海岸线长487公里。总面积18.8万平方公里。

河北省官方网站 http://www.hebei.gov.cn/
