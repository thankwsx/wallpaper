安徽位于华东腹地，是我国东部襟江近海的内陆省份，跨长江、淮河中下游，东连江苏、浙江，西接湖北、河南，南邻江西，北靠山东。安徽清初属江南省，康熙六年（公元1667年），析江南省为江苏、安徽两省而正式建省，取当时安庆、徽州两府首字得名。境内有皖山、皖水，即现今的天柱山和皖河，春秋时（公元前722-前481年）曾被封为伯国，称为皖国，故安徽简称皖。
