四川简称川或蜀，位于中国西南部，地处长江上游，素有“天府之国”的美誉。全省面积48.6万平方公里，辖21个市（州）、183个县（市、区），与重庆、贵州、云南、西藏、青海、甘肃和陕西等7省（自治区、直辖市）接壤，有全国最大的彝族聚居区、第二大藏区和唯一的羌族聚居区。2019年末全省户籍人口9099.5万人、常住人口8375万人，其中少数民族人口599.4万人。
